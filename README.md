# AetherSync-AI
AetherSync is a self-governing Multi-Agent Governance System aimed at solving the "AI Power Paradox". 

### Usage

Setup credentials

```
secrets
- .env
- credentials.json
- token.pkl
```

Installations
```
uv sync
```
*requirements.txt is just for reference

FastAPI
```
cd src
python main.py
```

MCP server
```
cd src
python mcp_server/server.py
```
